# Система управления библиотекой

## Описание
Это консольное приложение на языке Java для управления библиотекой. Все данные сохраняются в оперативной памяти программы во время ее работы.

Автор: Ару Сейткамал

## Описание файлов
Проект состоит из следующих файлов, каждый из которых выполняет свою задачу:

- `src/model/Book.java` - отвечает за хранение данных о книге (ID, название, автор, год издания).
- `src/model/Reader.java` - отвечает за хранение данных о читателе (ID, имя, телефон).
- `src/model/BorrowRecord.java` - отвечает за хранение записи о том, кто, какую книгу и когда взял.
- `src/service/LibraryService.java` - отвечает за всю логику работы программы: добавление книг, оформление выдачи и возврата, поиск, а также подсчет статистики.
- `src/main/Main.java` - отвечает за запуск программы, показывает текстовое меню в консоли и принимает команды пользователя.

## Как запустить проект в IntelliJ IDEA или VS Code

**Вариант 1: Запуск через IntelliJ IDEA (Рекомендуется)**
1. Откройте IntelliJ IDEA.
2. Нажмите **Open** и выберите папку с этим проектом.
3. Слева в панели файлов (раздел Project) откройте путь: `src` -> `main` -> `Main.java`.
4. Нажмите зеленой кнопкой Play слева от строки `public static void main` внутри файла `Main.java` и выберите **Run 'Main.main()'**.
5. Внизу экрана откроется консоль (терминал), где появится меню программы.

**Вариант 2: Запуск через Visual Studio Code (VS Code)**
1. Откройте VS Code.
2. Нажмите **File -> Open Folder...** и выберите папку с проектом.
3. Убедитесь, что у вас установлено расширение **Extension Pack for Java** (если нет — скачайте его во вкладке Extensions).
4. В панели слева откройте файл `src/main/Main.java`.
5. Нажмите кнопку **Run**, которая появится прямо в коде над надписью `public static void main`.
6. В терминале внизу экрана запустится программа и покажет главное меню.
